# 💻 Laptop Sales Power BI Dashboard

## 📌 Project Overview
This project presents an **interactive Power BI dashboard** created using a cleaned laptop sales dataset.  
The dashboard provides insights into laptop specifications, pricing patterns, storage distribution, and brand-level analysis to support data-driven decision making.

---

## 📂 Project Files
| File Name | Description |
|----------|------------|
| `Laptop Sales Dashboard.pbix` | Power BI dashboard file |
| `cleaned_laptop_sales_cleaned_dataset.csv` | Cleaned dataset |
| `README.md` | Project documentation |

---

## 🛠 Tools & Technologies
- **Power BI Desktop**
- **CSV Dataset**
- **DAX (Data Analysis Expressions)**
- **Power Query**

---

## 📊 Dataset Columns
The dashboard is built using the following columns:
- Company
- TypeName
- Disp_size
- IPS_Panel
- Touch_Screen
- Processor
- RAM_GB
- SSD_Disk
- HDD_Disk
- HybridFlash_Disk
- GPU_company
- Op_Sys
- Weight_kg
- Price_INR
- PPI
- SSD_storage
- HDD_storage
- Other_storage

---

## 📈 Dashboard Features
- KPI Cards:
  - Total Laptops
  - Average Price (INR)
  - Maximum Price
  - Average Weight
- Laptop count by company
- Average price comparison by brand
- RAM size distribution
- Storage type analysis (SSD, HDD, Other)
- GPU company comparison
- Laptop type (Notebook, Gaming, Ultrabook, etc.)
- Interactive slicers for:
  - Company
  - Processor
  - RAM
  - Touch Screen
  - IPS Panel
  - Operating System

---

## 🚀 How to Use
1. Download the file **`Laptop Sales Dashboard.pbix`**
2. Open it using **Power BI Desktop**
3. Click **Refresh** to load the dataset
4. Use slicers and visuals to explore insights
